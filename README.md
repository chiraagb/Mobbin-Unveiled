# Mobbin Unveiled

A Chrome extension that reveals blurred pro screens on [Mobbin.com](https://mobbin.com).

## Installation

1. Clone the repo and install dependencies:

   ```sh
   git clone https://github.com/chiraagbhandakkar/mobbin-unveiled
   cd mobbin-unveiled
   npm install
   ```

2. Build the extension:

   ```sh
   npm run build
   ```

3. Load in Chrome:
   - Go to `chrome://extensions/`
   - Enable **Developer mode**
   - Click **Load unpacked** → select the `dist/` folder

## Usage

Once installed, navigate to any app on Mobbin.com. The extension automatically reveals blurred pro screens as you scroll.
